Hour 0

Initial Setup

This workshop requires the Clojure and ClojureScript programming languages. We will be using Leiningen to create the project skeleton.

  1. Install Clojure and Leiningen.
  2. Open a shell.
  3. Navigate to the directory where you want to put the project. For example, if you want the project in ~/projects/project, you would cd ~/projects/.

  4. lein new app <project>

    In this workshop, we will be naming the project census. Therefore, the command would be lein new app census.

  5. Take a look around at files. Most notably:
    • project.clj
      • This is like pom.xml in a Maven project. It tells Leiningen where everything is.
      • Notice how there is a :main entry. This tells Leiningen which Clojure namespace to load when starting up this program. It looks for a -main function.
    • src/census/core.clj
      • Remember how in project.clj the :main entry was census.core? Well, that points to this file!
      • Find the -main function. Right now, it just prints a line that was put there as a stub by Leiningen. We'll be changing this later.
  6. Notice all those funny parentheses. Those are S-expressions, which are the most visible characteristic of Lisp dialects. We will be talking about those later.
  7. Back at the shell, in the census directory, run lein run. This will execute the program. Note that it prints the line in the function you saw earlier in src/census/core.clj.
