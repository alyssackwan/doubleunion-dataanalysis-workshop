Double Union Data Analysis Workshop

Data Analysis and Visualization with Clojure / Incanter and ClojureScript / D3

Download as .zip Download as .tar.gz View on GitHub

This workshop illustrates how to put together a data analysis and visualization application from the ground up. This includes the use of functional programming with some light discussion of using multiple concurrent processes.

The following tools are used:

  • Clojure programming language (back end)
    • Incanter library (data analysis)
    • Ring and Compojure libraries (web server)
  • ClojureScript programming language (front end)
    • D3 library (data visualization)
